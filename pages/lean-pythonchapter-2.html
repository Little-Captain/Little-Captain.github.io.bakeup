元素和切片

  • slice 访问格式 [startindex:endindex], 包括 startindex 的元素, 而不包括 endindex 的元素
  • [:endindex] 从 0 开始
  • [startindex:] 到 末尾 结束
mylist = ['a', 'b', 'c', 'd', 'e', 'f']
mylist[0] # 'a'
mylist[1:3] # ['b', 'c']

mylist = [1, 2, 3, ['a', 'b', 'c'], 5]
# 使用多重索引访问嵌套的序列
mylist[3][1] # 'b'

  • 字符串不可变
mystr = 'Paddington Station'
mystr=mystr.upper()
  • 赋值, 字符串成对使用 ' 或 " 来包裹. 你可以在一对中使用另一对
text = 'Paul said, "Hello World!"'
print(text)
  • 访问子串
text='Paul said, "Hi"'
text[:4]
text[-4:]
text[5:9]
text[0:4] + text[12:14]

'X'>'t'

  • 特殊字符和转义符

# % 操作符提供了字符串格式化功能, 结构如下
formatstring % (arguments to format)
# formatstring 包含 %标记的占位符
%c # 单字符或只有一个字符的字符串
%s # 字符串
%d # 整数
%f # 浮点数
%% # 百分号

ntoys = 4
myname='Fred'
'%s has %d toys' % (myname,ntoys)
# 'Fred has 4 toys'

lstrip() # 移除左边的空白字符
strip() # 移除左右两边的空白字符

mylist = []
names=['Tom','Dick','Harry']

names[1] # 'Dick'
elist[3][1] # 5

mylist = []

mylist.append('Tom')

mylist.append('Harry')
# 改变值
mylist[1]='Bill'
# 删除
del mylist[1] # # ['Tom','Harry']
  • 获取索引
mylist=['Tom','Dick','Harry']
mylist.index('Dick') # 1
mylist.index('Henry') # 不存在, 抛出错误
