Sequence

  • 有序
  • 通过索引定位
  • 切片操作取出子 sequence
  • String、Tuple(至少含有两个元素, (1) 为 int 1) 不可变(创建后, 元素不可变), List 可变
  • + * [] 是所有 sequence 通用的操作符
  • + 连接
  • * 重复序列, 产生新的序列
  • [] 取出指定的元素和子 sequence
# +
[1, 1] + [1, 2] # [1, 1, 1, 2]
(1, 2) + (1, 2) # [1, 2, 1, 2]
'12' + '12' # '1212'
# *
[1, 2] * 3 # [1, 2, 1, 2, 1, 2]
(1, 2) * 3 # (1, 2, 1, 2, 1, 2)
'12' * 3 # '121212'
# []
[1, 2, 3, 4][1:3] # [2, 3]
(1, 2, 3, 4)[1:3] # (2, 3)
'12312312'[1:3] # '23'

clear()
pop(key, [default]) # key 对应的值不存在, 返回 default 指定的默认值
